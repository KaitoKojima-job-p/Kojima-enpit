# 「ヒマナウ」 
![appstore](https://github.com/enpit2su-ics/2023-team-B/assets/146358551/52b89cf1-d03b-449f-ad11-a054d80377ab)  
空き時間の予定を共有し、スムーズに遊びに誘えるアプリ 

# エレベータピッチ
［友人との遊び計画の苦労］を解決したい  
［その日の気分で遊びを決める大学生］向けの  
［空き時間だけの予定表の共有］を実現する  
［ヒマナウ］です。  
これは［友達の空いている時間を確認して、遊びに誘う事］ができ、  
［Googleカレンダー］とは違って、  
［暇時間のみの共有であるため、プライバシーが守られ、共有画面がシンプルで、誰とでも共有しやすい環境］が備わっています。

# 想定利用者像
- 友達の空き時間を確認してから、遊びに誘いたい人
- 自分からは誘えないけど、遊びに誘ってもらいたい人


# フライヤー用
空き時間の予定を共有して、スムーズに遊びに誘おう！！！！！　　  
新感覚カレンダーアプリ『ヒマナウ』は、自分の予定を公開せずに、空いている時間帯だけを共有するアプリです。  
友達の空き時間を確認して遊びに誘おう。自分からは誘えなくても『ヒマナウ』を使っていれば、誰かが遊びに誘ってくれるかも。  
友達画面は、暇時間が長い順に友達を並び変えられたり、コミュニティ毎に友達を分類して表示できるなど自由度が高いです。  
誘う時は、メンバー選択→遊びジャンル選択→時間帯入力の3ステップ！  
誘われた時は、イベントグループに「参加する」を選択すれば完了。  
断りたい場合は、何もしなくて大丈夫。デフォルトでは、遊び開始30分前までにグループに入っていない人は、参加しないメンバーに自動的に振り分けられます。  
さあ、あなたも新感覚カレンダーアプリ『ヒマナウ』を使ってヒマな人を見つけよう！！

# プロダクトの説明
「ヒマナウ」は友達を遊びに誘うことをサポートするアプリ機能が備わっています。

- 暇時間登録機能
 → 自分が暇な時間帯を設定することで、暇時間を登録できます。暇時間は友達から常に閲覧できる状態にあり、友達はあなたの暇時間を見て、あなたを遊びに誘います。
  - ホーム画面では、自分が登録した暇な時間帯がカレンダーにガントチャートで表示されます。ガントチャートをタップすると暇な時間帯の詳細を確認できます。
  - 誤って暇な時間帯を登録してしまった場合でも、暇な時間帯を削除できます。
  - <カレンダー同期機能>
    - iPhone標準カレンダーの情報を取得し、予定の無い時間帯を暇時間としてガントチャートに表示します。
   
- 友達登録機能
 → 友達に登録することによって、遊びの誘いを送り合ったり、暇な時間帯を相互に閲覧することができます。
  - <ユーザーIDによる登録>
    - 友達のユーザーIDを入力し、表示された友達の名前をタップすることで友達に登録することができます。
  - <QRコードによる登録>
    - 友達の画面に表示されたQRコードを読み取ることで、友達に登録することができます。
   
- 友達の暇時間閲覧機能
 → 友達の暇時間がガントチャートで表示されており、友達を遊びに誘う時の参考にすることができます。
  -　友達の暇な時間帯は青色のバーで表示されており、左右にスクロールして閲覧できます。

- イベント勧誘機能
 → 友達の暇時間が表示されているガントチャートを参考に暇な友達を見つけ、誘うことができます。「イベント」とは友達同士が参加する遊びや予定のことを言います。
  - 誘う画面では、友達を選択した状態でカバンのマークをタップすることで友達をイベントに誘うことができます。誘った友達とあなたの名前がメンバーとして表示されます。
  - イベントを作成すると、誘った友達のイベント画面にあなたが作成したイベントが表示されます。
 
- イベント参加機能
 → イベント画面には友達から誘われたイベントとイベントの詳細な情報が表示され、「参加する」ボタンをタップすることでイベントに参加する意思をイベントメンバーに示せます。
  - イベントに参加する・しないを選択でき、「参加する」ボタンの下の締切時刻を表すゲージが0になると自動的にイベントに参加しないを選択します。
  - メンバーの一覧ではイベントの作成者は青色で表示され、参加しているメンバーは黒くハイライトされ、参加していないメンバーは灰色で表示されます。
  - イベントに参加した後に急な予定で参加できなくなった場合でも、「退会する」ボタンでイベントから抜けることができます。
  - <チャット画面>
    - イベントの作成者とイベントに誘われたメンバー同士でチャットを相互に送信・閲覧できます。
 
- プロフィール登録機能
 → あなたの名前やアイコン写真などのプロフィールをカスタマイズできます。
  - 自分のユーザーIDが表示されており、これを元にあなたの友達は友達登録が行えます。

- 予定追加機能
    -  同じ予定の繰り返し設定
    -  履歴から同じ時間、タイトルの予定を入力
    -  予定ごと色によるラベル分け
    - 暇時間のみの予定表共有機能
    - 友達を所属するコミュニティごとに分類して、表示
    - 暇な時間が長い順にソートして友達の暇時間を表示
